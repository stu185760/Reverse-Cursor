Debug Ad Creation

Open browser console (F12) and check the logs when creating an ad.

Look for these log messages:

  • [createAd] Original ad data: ...
  • [createAd] Status validation: ...
  • [createAd] Final ad data being inserted: ...
  • [createAd] Database error details: ... (if error occurs)

Test Data

Try creating an ad with these values:

  • Category: Jewelry
  • Title: Test Ad for Debugging
  • Description: This is a test ad to debug the constraint issue
  • Location: Mumbai
  • Min Budget: 1000
  • Max Budget: 5000
