## Quick Guide

You need to add PNG icons in these sizes to this folder:

- `icon-72x72.png`
- `icon-96x96.png`
- `icon-128x128.png`
- `icon-144x144.png`
- `icon-152x152.png`
- `icon-192x192.png`
- `icon-384x384.png`
- `icon-512x512.png`

## Easiest Method:

1. Go to: **https://favicon.io/favicon-generator/**
2. Settings:
   - Text: "EC" or "E"
   - Background: #0D0D0D
   - Font Color: #00FFFF
   - Font: Bold
3. Download ZIP
4. Extract all PNG files to this folder
5. Rename to match the sizes above

## Or Use:

- **https://www.pwabuilder.com/** (PWA specific tool)
